# personal-config

## 说明

这个仓库用来存储平时使用的工具和配置，在修改这个仓库的时候务必要遵守下列规则.

1. 配置适应的环境需要做出说明
2. 配置需要尽量通用，兼容多平台多版本操作系统
3. 避免提交未验证过的软件配置和陌生软件配置
4. 配置项尽量添加注释
